Packed Node format is a way to store the node info in a small yet easy to parse format.

It is used in many places in Tox, e.g. in DHT Send nodes.

To store more than one node, simply append another on to the previous one:

[packed node 1][packed node 2][...]

Packed node format:

                         (39 bytes for IPv4, 51 for IPv6)
+-----------------------------------+
| ip_type                ( 1 byte ) |
|                                   |
| IPv4 Address           ( 4 bytes) |
|  -OR-                     -OR-    |
| IPv6 Address           (16 bytes) |
| Port                   ( 2 bytes) |
| Node ID                (32 bytes) |
+-----------------------------------+

DHT module should use only UDP variants of IpType, given that DHT runs solely on the UDP.
