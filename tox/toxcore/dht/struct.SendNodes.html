Response to GetNodes request, containing up to 4 nodes closest to the requested node.

Packet type 0x04.

Serialized form:

Length Contents
1 Number of packed nodes (maximum 4)
[39, 204] Nodes in packed format
8 Ping ID

An IPv4 node is 39 bytes, an IPv6 node is 51 bytes, so the maximum size is 51 * 4 = 204 bytes.

Serialized form should be put in the encrypted part of DHT packet.

Fields

nodes

Nodes sent in response to GetNodes request.

There can be only 1 to 4 nodes in SendNodes.

id

Ping id that was received in GetNodes request.
